Im folgenden Spiel soll eine Entscheidung über die Durchfürung einer politischen Reform getroffen werden. Für sie und ihren Mitspieler entstehen potentiell Kosten aus dieser Reform. Mit einer warscheinlichkeit von {{ Constants.p }} kostet sie die durchführung der Reform {{ Constants.cbar }} Taler. Mit einer warscheinlichkeit von {{ Constants.prev }} kostet sie diese Reform nichts. Ihr Mitspiel befindet sich in der selben Situation wie sie. Es stehen {{ Constants.v }} Taler zur Verfügung um sie beide zu kompensieren.

Im folgenden werden ihnen ihre durch die Reform entstehenden Kosten mitgeteilt. Dannach können sie eine Kompensation in Höhe von 20 Talern für die ihnen entstandenen Kosten fordern. Im Fall das die Summe der geforderten Kompensation den gesamtgesellchaftlichen Nutzen übersteigt wird die Reform nicht umgesetzt und Sie und Ihr Mitspieler bekommen eine Auszahlung von 0 Talern. Im Fall das die geforderte Kompensation den gesellschaftlichen Nutzen nicht übersteigt wird zuerst der Spieler der eine Kompensation gefordert hat in Höhe von 20 Talern kompensiert. Das verbleibende Budget wird an den anderen Spieler ausgezahlt. Fals keiner der beiden Spieler eine Kompensation gefordert hat bekommt jeder 15 Taler ausgezahlt.
