Ist die folgende Aussage wahr oder falsch?

Falsch! "Sie haben kosten von 0 und sagen ihrem Mitspieler das sie Kosten von 0 haben. Ihr Mitspieler sagt ihnen das er Kosten von 0 hat. Die Reform wird durchgeführt und sie bekommen eine Auszahlung von 20." , ist eine wahre Aussage. Wenn ihr Mitspieler keine Kosten von 0 angibt wird die Reform durchgeführt und sie erhalten ihre gewünschte Kompensation in Höhe von 20 Talern. Da die Reform ihnen keine Kosten verursacht entspricht das ihrem Gewinn.
