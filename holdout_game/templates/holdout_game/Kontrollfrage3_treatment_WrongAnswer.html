Ist die folgende Aussage wahr oder falsch?

Falsch! "Sie haben Kosten von 0 und sagen ihrem Mitspieler das sie Kosten von 0 haben. Ihr Mitspieler fordert keine sagt ihnen das er Kosten von 0 hat. Die Reform wird durchgeführt und sie bekommen beide eine Auszahlung von 10." ist eine falsche Aussage. Da sie beide Kosten von 0 angegeben haben wird die Reform durchgeführt. Ihre Auszahlung beträgt allerdings 15 da sie Kosten von 0 haben und einen Transfer von 15 bekommen.
